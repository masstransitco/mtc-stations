### Dataset: Distribution of Metered Parking Spaces and Occupancy of those Installed with New Parking Meters


#### Data Type Overview


| Name          | Type          | Description                        |
|---------------|---------------|------------------------------------|
| parkingspaces | Spatial Layer | Metered parking space              |


***


#### Attribute Descriptions


| Field Name         | Data Type  | Null Option | Description                                                                                                   |
|--------------------|------------|-------------|---------------------------------------------------------------------------------------------------------------|
| OBJECTID           | OID        | Not Null    | Object identifier generated by system automatically                                                           |
| PoleId             | Long       | Nullable    | Identifier number of the meter pole                                                                           |
| ParkingSpaceId     | String     | Nullable    | This value specifies the metered parking space                                                                |
| Region             | String     | Nullable    | Region name (English)                                                                                         |
| Region_tc          | String     | Nullable    | Region name (Traditional Chinese)                                                                             |
| Region_sc          | String     | Nullable    | Region name (Simplified Chinese)                                                                              |
| District           | String     | Nullable    | District name (English)                                                                                       |
| District_tc        | String     | Nullable    | District name (Traditional Chinese)                                                                           |
| District_sc        | String     | Nullable    | District name (Simplified Chinese)                                                                            |
| SubDistrict        | String     | Nullable    | Sub-District name (English)                                                                                   |
| SubDistrict_tc     | String     | Nullable    | Sub-District name (Traditional Chinese)                                                                       |
| SubDistrict_sc     | String     | Nullable    | Sub-District name (Simplified Chinese)                                                                        |
| Street             | String     | Nullable    | Street Name (English)                                                                                         |
| Street_tc          | String     | Nullable    | Street Name (Traditional Chinese)                                                                             |
| Street_sc          | String     | Nullable    | Street Name (Simplified Chinese)                                                                              |
| SectionOfStreet    | String     | Nullable    | Section of the Street (English) where the parking meter is installed                                          |
| SectionOfStreet_tc | String     | Nullable    | Section of the Street (Traditional Chinese)                                                                   |
| SectionOfStreet_sc | String     | Nullable    | Section of the Street (Simplified Chinese)                                                                    |
| VehicleType        | String     | Nullable    | Type of vehicle of the metered parking space                                                                  |
| LPP                | Long       | Nullable    | The longest parking period (minutes) that could be purchased at one time                                      |
| OperatingPeriod    | String     | Nullable    | Operating hours of the metered parking space                                                                  |
| TimeUnit           | Long       | Nullable    | Minimum parking time (minutes) to be purchased                                                                |
| PaymentUnit        | Double     | Nullable    | Price of one parking time unit                                                                                |
| Shape              | Geometry   | Not Null    | Geometry (coordinates - spatial data)                                                                         |
| VacancyUrl         | String     | Nullable    | VacancyUrl (link for occupancy status)                                                                        |

#### Other Details


- Multilingual support: Region, district, sub-district, street, and section fields in English, Traditional Chinese, and Simplified Chinese.
- Fields support both spatial (GIS) and non-spatial queries.
- **VacancyUrl** allows integration with real-time vacancy queries for each site.
- **Shape** provides geometry for mapping and analysis.


***


This full specification provides all field definitions, coded values, and schema requirements as structured in the official dataset for metered parking spaces in Hong Kong[1].
